# Credit Card Financial Dashboard

## Overview

The **Credit Card Financial Dashboard** project analyzes credit card transactions and customer data to generate meaningful financial insights.
Using **Power BI**, the dashboard visualizes transaction patterns, customer demographics, revenue metrics, and spending behavior to help stakeholders make data-driven decisions.

This project demonstrates the use of **data integration, KPI tracking, and interactive visualization** for financial analytics.

---

## Tools & Technologies

* **Power BI**
* **PostgreSQL**
* **Excel**
* Data Visualization
* Financial Data Analysis

---

## Dataset

The project uses multiple datasets to analyze credit card usage and customer behavior.

Main datasets include:

* `credit_card.csv` – transaction details
* `customer.csv` – customer demographic data
* `cc_add.csv` – additional credit card information
* `cust_add.csv` – additional customer details

These datasets were combined to create a **comprehensive financial analytics dashboard**.

---

## Dashboard Features

The Power BI dashboard provides insights into:

* Total revenue generated from credit card transactions
* Customer spending behavior
* Transaction volume trends
* Customer demographics
* High-value customer segments
* Financial performance KPIs

---

## Key Insights

Some important insights derived from the dashboard:

* Identification of **high-spending customer segments**
* Detection of **transaction trends and seasonal spending patterns**
* Analysis of **revenue contribution by different customer groups**
* Understanding **customer demographics affecting spending behavior**

These insights help businesses improve **financial planning and customer targeting strategies**.
